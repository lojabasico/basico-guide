# Tamanhos de imagens

**importante!**  

Ao exportar a imagem, usar sempre **Save for Web**!  
Photoshop > File > Export > Save For Web > ver qual o melhor modo de compressão.  

**Dica do Renato:** 2-UP > .jpeg > escolher a qualidade (sempre visualizar a 200%).
